Stein is a small library that contains functions that are fairly common to projects at [SmartLogic](https://smartlogic.io).

## Installation

You can install Stein by adding it to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:stein, "~> 0.1.0"}
  ]
end
```
